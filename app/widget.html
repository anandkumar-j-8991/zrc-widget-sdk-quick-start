ZOHO.CRM.CONNECTOR

Connector APIs allow you to authorize and invoke pre-configured Connectors from the widget. Use authorize to prompt the connector auth window, and invokeAPI to call a connector endpoint.

authorize(nameSpace)

Prompts the Connector authorization window. Returns a promise resolved with true on success.

Response

invokeAPI(nameSpace, data)

Invoke a Connector API. Pass dynamic data via the VARIABLES key, or use CONTENT_TYPE / PARTS / FILE for multipart uploads.

Response

invokeAPI — Multipart File Upload

Upload a file through a Connector API using multipart content type.

Response
